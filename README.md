# NAVIT — Site inicial

Este repositório contém um site estático inicial para NAVIT — Núcleo de Apoio à Vida.
Inclui:
- index.html — estrutura do site.
- styles.css — estilos responsivos.
- script.js — funcionalidades leves (galeria, menu mobile, botão WhatsApp).
- assets/logo.png — logo enviado pelo usuário (instruções no arquivo assets/logo-instructions.txt).

Como personalizar
1. Substitua `assets/logo.png` pelo logo oficial (SVG ou PNG) se quiser uma versão vetorial.
2. Coloque as fotos reais em `assets/images/` e atualize as URLs em `index.html`.
3. Atualize textos, horários, locais e contatos conforme necessário.
4. Para publicar: ative o GitHub Pages (Settings → Pages) apontando para a branch `main` (ou `gh-pages`).

Deploy rápido (opção manual)
- git init
- git add .
- git commit -m "Site NAVIT inicial"
- criar repositório no GitHub e push para main
- Ativar Pages no GitHub
