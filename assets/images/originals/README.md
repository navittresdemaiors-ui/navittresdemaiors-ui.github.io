# Pasta de originais - NAVIT

Coloque aqui os arquivos de imagem originais que você deseja manter no repositório. Recomenda-se manter apenas cópias de backup e versões otimizadas separadas em `assets/images/`.

Instruções rápidas:
- Faça upload das imagens originais nesta pasta (por exemplo: `encontro-01.jpg`).
- Evite arquivos muito grandes (>10 MB). Se forem grandes, considere reduzir antes.
- Quando quiser que eu gere as versões otimizadas (WebP 1200/800/400, thumbnails), me avise com a lista de nomes.
- Para cada imagem enviada, eu posso:
  - salvar o original em `assets/images/originals/`,
  - gerar variantes otimizadas: `photo-01-1200w.webp`, `photo-01-800w.webp`, `photo-01-400w.webp`, `photo-01@2x.webp`, `photo-01-thumb.webp`,
  - atualizar a galeria automaticamente.

Legenda/ano:
- Se souber o ano de cada foto, envie uma lista (ex.: `photo-01 = 2018`) para eu já inserir nas legendas. Caso contrário, uso `unknown`.
